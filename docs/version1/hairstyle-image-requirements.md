# GroomAI Hairstyle Image Requirements

This document lists all 50 hairstyles currently programmed into the GroomAI Barber Translator feature. The design team needs to source or generate **one highly realistic, high-quality image** for each hairstyle.

## ⚠️ Important Naming Instructions
- Save all images as `.png` or `.jpg` using **EXACTLY** the filenames provided below (they must match the `slug` perfectly).
- Place all finalized images in the `assets/images/hairstyles/` folder.
- Images should ideally be vertically oriented (portrait) and focus clearly on the hair.
